FaceMagic — 使用说明
════════════════════════════════════════════

一、首次使用
──────────────────────────────────────────
1. 双击「一键启动.bat」
2. 等待环境自动配置（约 500MB 下载，耗时看网速）
3. 看到 "环境配置完成" 后自动弹出登录窗口
4. 输入管理员给你的账号密码登录
5. 登录成功后进入主界面开始使用

注意：首次配置只需一次，之后双击「一键启动.bat」直接进

二、系统要求
──────────────────────────────────────────
- Windows 10/11
- Python 3.10 ~ 3.12（如未安装，下载地址如下）
  https://www.python.org/downloads/
  安装时记得勾选 "Add Python to PATH"

三、可选配置（有 NVIDIA 显卡时）
──────────────────────────────────────────
如需 GPU 加速，需额外安装：
1. NVIDIA 显卡驱动（最新版）
2. CUDA Toolkit 12.x
   https://developer.nvidia.com/cuda-downloads
3. cuDNN 对应版本
安装后「一键启动.bat」会自动使用 GPU，速度翻几倍

没有 NVIDIA 显卡也能用，会自动切换 CPU 模式

四、验证服务器说明
──────────────────────────────────────────
此程序需要连接验证服务器登录使用。
如需修改服务器地址，编辑 auth_config.json 中的 server_url。

五、常见问题
──────────────────────────────────────────
Q: 启动后闪退？
A: 在「一键启动.bat」上右键→编辑，确认 Python 路径正确

Q: 提示模块找不到？
A: 删除 venv 文件夹，重新运行「一键启动.bat」

Q: 登录提示无法连接服务器？
A: 检查 auth_config.json 里的 server_url 是否正确
